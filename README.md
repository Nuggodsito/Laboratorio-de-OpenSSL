# Laboratorio OpenSSL - Seguridad en PHP

## Integrantes
- Luis Calderón
- Luis Ortega

## Resumen del Laboratorio
Implementación de funciones criptográficas usando la extensión OpenSSL en PHP. Se trabajó con firmas digitales, verificación de autenticidad y encriptación de datos mediante ejemplos prácticos y documentación técnica.

## Aprendizajes Principales
- Generación y uso de claves públicas y privadas RSA
- Creación y firma de certificados X.509
- Implementación de firmas digitales con SHA256
- Cifrado y descifrado con AES-128-CBC
- Verificación de integridad de datos mediante criptografía

## Tecnologías Utilizadas
- PHP con extensión OpenSSL
- Algoritmos: RSA, AES, SHA256
- Formatos: PEM, CSR, X.509
